# 🪥 PureBrush - Landing Page

Esta é uma landing page moderna e responsiva para a **PureBrush**, uma marca focada em higiene e estética bucal. O projeto foi desenvolvido como parte dos meus estudos de desenvolvimento Web, focando em semântica HTML, estilização avançada com CSS e responsividade.

## 🚀 Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização com Flexbox, efeitos de transição e fontes externas (Google Fonts - Poppins).
* **Responsividade:** Implementação de `@media queries` para adaptação em dispositivos móveis.

## 📱 Funcionalidades

* **Menu de Navegação:** Navegação suave (*smooth scroll*) para diferentes seções da página.
* **Seções de Benefícios:** Layouts alternados com imagens e textos explicativos.
* **Galeria de Produtos:** Exibição em grid dos produtos "Best-sellers".
* **Integração Social:** Link direto para o Instagram da marca.
* **Design Responsivo:** Layout totalmente adaptado para telas menores (smartphones).

## 🛠️ Como executar o projeto

Para visualizar o projeto localmente, siga os passos abaixo:

1. Clone o repositório:
   **Bash**

   ```
   git clone https://github.com/seu-usuario/purebrush.git
   ```
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em seu navegador de preferência.

## 📂 Estrutura de Arquivos

* `index.html`: Estrutura principal da página.
* `style.css`: Estilização completa e regras de responsividade.
* `/img`: Pasta contendo as imagens utilizadas no layout.
